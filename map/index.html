Utilities Providing Energy Data for Benchmarking in ENERGY STAR® Portfolio Manager® Web Services Fact Sheet

Aggregate Whole-Building Data?

Does utility provide monthly energy data, aggregated across multiple tenants within a building? Numbers in parenthesis are (minimum number of tenants / maximum percent of building consumption attributable to any one tenant), these requirements determine if the utility will provide data without explicit tenant consent.

Multifamily Included?

Is this service available for owners/operators of multifamily properties, in addition to commercial buildings?

Data Type

Utility will provide aggregate data in spreadsheet format, or will use software called web services to send data directly into a customer's Portfolio Manager account.
